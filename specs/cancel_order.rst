.. _cancel_order:

cancel_order
=======================================
扩展进程 -> 主进程，撤销交易指令

Example
--------------------------------------------------
.. code-block:: javascript

  {
    "aid": "cancel_order",          //必填, 撤单请求
    "order_id": "0001",             //必填, 委托单的order_id
  }
    
Remarks
--------------------------------------------------
由扩展进程发往主进程，用于撤销一个交易指令
